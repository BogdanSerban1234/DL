P% = P/100 (Fractie ordinara)
EX 1. 75% = 75/100 = 0,75
EX 2. 0,3 = x%, x=? 0,3 = 3/10 = 30/100 = 30%
EX 3. 33% = 33/100 = 1/3 = 1:3 = 0,(3)

Tipuri de operatii cu procente

EX 1. Aflarea unui procent dintr-un numar
P% din X = P/100 * x
exemplu: 23% din 75 = 23/100 * 75= 23/100 * 75/1 = 23*75/100 = 1725/100 = 17,25

EX 2. Aflarea unui nr cand stim un procent din el
P% din X = a, x = ? x = a : p/100 = 75 * 100/32 = 7500/32 (simplificam cu 4) = 1875/8 = 234,375

EX 3. Aflarea unui procent cand cunoastem un nr si procentul din el
Avem formula P% din a = b, P% = ? => P% * a = b => P/100 * a = b => P/100 = b/a => P = b*100/a
exemplu: x% din 15 = 180, x = ? x/100 * 15 = 180 => 180*100/15 => (simplificam cu 15) 12*100/1 => 1200%

Exista si cresteri si scaderi in procente, pentru a face asta trebuie sa adaugati/scadeti procentul din nr la nr.
exemplu: Un caiet costa 2,5 lei, cat costa dupa o scumpire de 10%? x + 10% din x = 110% din x = 110/100 *2,5 = 110*2,5/100 = 275/100 = 2,75 lei
