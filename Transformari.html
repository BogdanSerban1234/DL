Aceasta lectie are nevoie de cunostinte de la lectia de Fractii

Transformarile in fizica sunt practic doar fractii, dar unpic diferit, avand multe formule.

In fizica, am invatat ca avem multe tipuri de marimi fizice:

  • Arie
  • Volum
  • Distanta
  • Viteza
  • Timpul

Pentru toate acestea, se pot face numeroase transformari, explicate in fata

1. Aria este M * M (* este ori, ex 5 * 2 = 10), Aria se noteaza cu puterea a 2-a, si se poate transforma in arie prin -> m * m = m²

2. Volumul este M * M * M, Volumul se noteaza cu puterea a 2-a, si se poate transforma in volum prin -> m * m * m= m³

3. Distanta

  1. Def: Lungimea traictoriei parcursa de un mobil se numeste distanta
  2. Traictoria poate fi:
    • Rectilinie (un mobil ce se deplaseaza intr-o linie dreapta se numeste Rectilinie)
    • Curbilinie (un mobil ce se deplaseaza intr-o linie curba se numeste Curbilinie)
  3. Unitatea de masura in SI este metrul, alte unitati de masura fiind Km (1000m) cm(0.01m) etc.
  4. Dedus: D in SI = V in SI * T in SI = m/s * s = m Direct: D in si = m
  5. Expresia matematica: D = V * T
  6. Explicatie: M= metru, V = viteza cu care merge mobilul,D = distanta parcursa de mobil, T = timpul
  7. Transformarile se fac asa => D = 25m/s * 5s => D = 25m * 5 = 125m

4. Viteza

  1. Def: Viteza este o marime fizica numeric egala cu Distanta parcursa in unitatea de timp
  2. Def 2: m/s reprezinta viteza unui mobil atunci cand mobilul se deplaseaza pe o distanta de un metru timp de o secunda
  3. Unitatea de masura in SI este metrul pe secunda, alte unitati de masura fiind Km/h (1000m/3600s) etc.
  4. Dedus: V in SI = D in SI / T in SI = M/S (Metru/Secunda) Direct: V in SI = m/s
  5. Expresia matematica: V = D/T
  6. Explicatie: V = viteza cu care merge mobilul,D = distanta parcursa de mobil, T = timpul
  7. Transformarile se fac asa => 20m/s = 20/1000 (transformare in KM) supra 1/3600 (Transformare in ore) = 20/1000 : 1/3600 = 20/1000 * 3600/1 = 20*3600/1000 = 72000/1000 = 72/1 = 72 km/h

5. Timp

  1. Def: Nu a fost notata o definitie la ore
  2. Unitatea de masura in SI este secunda , alte unitati de masura fiind ora (3600s), minutul (60s) etc.
  3. Dedus: T in SI = D in SI / V in SI = M/M pe secunda = s Direct: T in SI = s
  4. Expresia matematica: T = D / V
  5. Explicatie: V = viteza cu care merge mobilul,D = distanta parcursa de mobil, T = timpul in care este parcursa distanta
  6. Transformarile se fac asa => 15 h = 3600s * 15 = 54000s
