  2. Forta este o marime fizica vectoriala ce exprima cantitativ interactiunea dintre 2 corpuri
  3. Expresia matematica: F = m*a
  4. Explicatie:
    5. F-Forta

    a- acceleratia
  5. Unitatea de mas in SI
    1. dedus

      [F]si = [m]si * [a]si = kg * m/s2 = N (newton)
    2. direct
      3. [F]si = N (newtonul)
  6. Def u de mas in SI
    7. un Newton reprezinta o forta ce actioneaza asupra unui corp cu masa de un kg si careia ii imprima o aceleratie de un m/s2
