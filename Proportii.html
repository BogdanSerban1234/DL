Aceasta lectie are nevoie de cunostinte de la lectia de Fractii

Proportiile sunt fractii egale. EX: 5/15 = 10/30

Proportiile derivate sunt proportii care au fost modificate:

  • Cu aceasi termeni
    • Cand termenii sunt modificati, se schimba intre ei mezi si extremii (Extremii Sunt stanga sus si dreapta jos, mezi sunt inversul)
  • Cu termeni modificati
    • Proportile se pot modifica cu alte numere, Mai jos sunt exemple:
      • a/b = c/d => a*m/b*m = c/d
      • a/b = c/d => a:m/b:m = c/d
      • a*m/b = c*m/d
        • Practic, trebuie doar sa inmultesti numerele cu acelasi nr. Asta merge si pentru Minus sau Plus doar ca trebuie adaugat unul din mezi sau extremi.

Avem ca exemplu pe a/b = c/d, Proporiile au proprietatea ca a*c = b*d (Mezii inmultiti sunt egali cu extremii inmultiti)

Avem doua tipuri de rezolvare de probleme:

  • Cu proportii derivate
    • a/b = 2/3 => a/a+b = 2/2+3 => a/23 = 2/5 => a = 23*2/5 = 46/5
      a/b = 2/3 => a+b/b = 2+3/3 => 23/b = 5/3 => b = 23*3/5 = 69/5
  • Metoda cu K
    • a + b = 100 a/6 = b/4 = k =>
      a = 5k |
      b = 4k | 5k + 4k = 100 => 100 = 9k => k = 100 : 10 = 10 (a = 60 , b = 40)

Cand se lucreaza cu puteri, vom folosi k la puterea n (a = 100 ,Ex. a la puterea 2 = (25k) la puterea 2 = 25*k(k la puterea 2) => 25k (k la puterea 2a) = 100 => k la a doua = 4 => k = 2)
